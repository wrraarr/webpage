    1. Length Matters Most
    A 16-character password with only lowercase letters is stronger than an 8-character password with all character types. Aim for at least 12-16 characters.
    2. Use a Password Manager
    Let software like 1Password, Bitwarden, or LastPass generate and store strong, unique passwords for each site. You only need to remember one master password.
    3. Passphrases Beat Complex Passwords
    "correct-horse-battery-staple" is stronger and more memorable than "P@ssw0rd1". Random words strung together create strong passwords.
    4. Avoid Common Substitutions
    Replacing 'a' with '@' or 'o' with '0' doesn't fool modern cracking tools. "P@ssw0rd" is just as weak as "Password".
    5. Never Reuse Passwords
    If one site gets breached, attackers will try that password everywhere. Every account needs a unique password.
    6. Enable Multi-Factor Authentication (MFA)
    Even if someone gets your password, MFA requires a second form of verification (usually your phone). This stops most attacks.
